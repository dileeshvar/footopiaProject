Rules
The rules of Footopia are as follows: The user can select a team of 11 members with
1 Goal Keeper
4 Defenders
4 MidFielders
2 Strikers
The user can set his team before the game week starts, which is usually Saturday.
The user is allowed {{allowed_transfers}} transfers per season/tournament.
The user is allowed {{wildcard}} wildcards per season/tournament.

The points system is as follows:


ScenarioPoints
Goalkeeper scores goal 5
Defender scores goal4
Midfielder scores goal3
Striker scores goal3
Player scores own goal-3
Player gets yellow card-1
Player gets red card-5
Player played minutes1*
*-1 point for every 45 minutes played
